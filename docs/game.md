# 게임 기획/설계

`dori` 게임의 컨셉·현재 상태·로드맵을 정리한 문서입니다. 게임이 발전하면 이 문서에 계속 기록합니다.

## 컨셉

- 정체성: **결정 돕기 · 랜덤 뽑기 · 복불복**을 모두 포괄하는 캐주얼 미니게임 **컬렉션 사이트**
- 방향: 세로형(모바일 우선), 720×1280 기준, 화면에 맞춰 `FIT` 스케일
- 조작: 탭(터치/클릭) 중심의 가벼운 플레이
- 메뉴(허브)에서 게임을 골라 즐기는 **미니게임 포털** 형태

### 포괄 범위 & 수록 예정 게임

- **결정 돕기**: 점심 메뉴 룰렛 · 사다리타기 · 랜덤 핀볼
- **랜덤 뽑기 / 추첨**: 룰렛·사다리 등을 추첨 용도로 활용 + 뽑기 상자류 확장
- **복불복**: 러시안 룰렛 · 악어 이빨 누르기 · 통아저씨(팝업 해적)

> 세 범주 모두 난수·확률 기반이라 공통 이론([game-theory.md](./game-theory.md))과 메커니즘([game-mechanics.md](./game-mechanics.md))을 공유한다.

## 씬 구조

```
Boot → Preload → Hub → (게임 선택) → 각 미니게임 → (⬅ 메뉴) → Hub
```

| 씬 | 역할 |
|----|------|
| `Boot` | 최초 초기화 후 Preload로 전환 |
| `Preload` | 로딩 바 표시 후 Hub로 전환 |
| `Hub` | 타이틀 + 범주별 게임 선택(미구현은 '준비 중') |
| `Roulette` 등 | 각 미니게임(`MiniGame` 베이스 상속) |

## 현재 상태

- **허브**: 결정 돕기 · 랜덤 뽑기 · 복불복 범주로 게임 목록 노출, 미구현은 '준비 중'으로 흐리게(정직한 어포던스). 타이틀 "dori"는 그라디언트·그림자·강조선·은은한 호흡 연출, 하단에 제작자 GitHub 크레딧.
- **점심 메뉴 룰렛(구현 완료)**: 결과를 RNG로 먼저 정하고 포인터 칸에 정확히 멈추는 **정직한 매핑**, `Cubic.easeOut` 감속, 결과 팝업, 다시 돌리기.
- 공통 토큰(`theme.js`)·어포던스 버튼(`ui.js`)·FSM 베이스(`MiniGame.js`) 적용.

## 룰렛 수치 메모

- 메뉴 8종, 원판 반경 300px, 중심 `(360, 560)`

- 팔레트 강조 4색(`SLICE`) 순환, 라벨은 배경색으로 대비
