# dori 🎡

> 뭐 먹지? 누가 하지? — **고민은 짧게, 결정은 즐겁게.**

**dori**는 결정이 필요한 순간을 위한 결정 도구 모음입니다.
룰렛을 돌리고, 사다리를 타고, 핀볼을 떨어뜨리면 — 남은 건 결과를 즐기는 것뿐.
속임수 없이 **화면에 보이는 그대로 뽑는 정직한 랜덤**이 원칙이고, 모든 게임은 오락 목적입니다.

결정 돕기 · 랜덤 뽑기 · 복불복 **도구 컬렉션**.

**▶ 플레이: https://dori.io.kr/**

## 도구 (10종)

| 범주 | 도구 | 핵심 |
|------|------|------|
| 결정 돕기 | 🍚 메뉴 룰렛 | 시간대(아침·점심·저녁·야식)별 트렌드 메뉴, 편집·저장, 숫자 모드, 나온 메뉴 제외 |
| | 🪜 사다리타기 | 참가자 2–6명, 용도 프리셋(커피·벌칙·청소·당첨), 색상 경로 |
| | 🎯 랜덤 핀볼 | 매판 랜덤 맵 + 맵 섞기, 칸 4–10 설정, 낙하 지점 선택 → 실제 물리 낙하 |
| | 👥 조 배정 | 번호 또는 이름 명단으로 N조 분할 — 참여·쉬기·조장 지정, 편성 복사 |
| 랜덤 뽑기 | 🎁 뽑기 상자 | 비복원 추첨 — 뽑힌 항목은 제외, 남은 풀이 항상 보임 |
| | 🎱 로또 추첨 | 최대 5줄, 매 정각 한 번 — 복사·공유 가능 |
| 복불복 | 💥 러시안 룰렛 | 6약실, 조건부 확률을 화면에 정직하게 표시 |
| | 🐊 악어 이빨 | 12개 중 함정 1개, 걸린 이빨은 빨간색으로 공개 |
| | 🗡️ 해적통 | 칼을 꽂다 보면… 펑! |
| | 🎴 단청(丹靑) | 붉·청 색패가 답하는 두 갈래 문답 — 어디서 물어도 4시간 동안 같은 답 |

## 설계 원칙

- **정직함이 최우선** — 화면에 보이는 것과 결과가 항상 일치(정직한 매핑·확률 공개), 다크 패턴 없음
- **게이미피케이션 + 어포던스**를 모든 결정의 기준으로 ([docs/design-principles.md](docs/design-principles.md))
- **색상 연결** — 결과의 색은 그 출처(칸·참가자·이빨)의 색을 따라감
- **시간대 감성** — 접속 시각에 따라 분위기·인사말·메뉴가 바뀜(서카디안 리듬 기반)
- **바로가기(PWA)** — 허브의 📲 버튼으로 홈 화면/바탕화면에 앱처럼 추가, QR·공유 버튼 내장

## 기술

- **Phaser 3** (CDN) + 바닐라 JS ES 모듈 — 빌드 도구 없음
- **GitHub Pages** 배포 — `main` 푸시 후 약 20초면 반영
- 에셋 대부분 코드 생성(도형·이모지 텍스처·Web Audio 합성음)

## 이용 안내

dori의 모든 도구는 **오락 목적**입니다. 결과는 재미를 위한 무작위(또는 결정적) 값일 뿐이며,
**금전·재물이 걸린 도박/내기, 사행 행위, 기만 등 비윤리적·불법적 목적에 사용하는 것을 금지**합니다.
