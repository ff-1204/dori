### 🎮 게임 7종

- **메뉴 룰렛** — 시간대(아침 05–11 / 점심 11–17 / 저녁 17–21 / 야식 21–05)에 맞는 트렌드 기반 메뉴 세트. 메뉴 추가·삭제(시간대별 독립 저장)와 기본값 복원, 숫자 룰렛 모드(1~N). 결과-우선 회전으로 포인터 칸과 결과가 항상 일치.
- **사다리타기** — 참가자 2~6명·결과 편집, 용도 프리셋(커피/벌칙/청소/당첨). '시작' 시 다리가 전부 공개되고 참가자 색으로 경로가 그려짐.
- **랜덤 핀볼** — 낙하 지점을 직접 고르고(주도성) 실제 물리로 낙하. 핀 히트 카운터, 막판 슬로모+줌, 착지 파티클. 칸 라벨 편집(기본 1·2·3등+꽝).
- **뽑기 상자** — 비복원 추첨. 뽑힌 항목은 흐려져 남은 풀이 항상 보임. 항목 2~12개 편집.
- **러시안 룰렛** — 6약실, 시도마다 실제 확률(1/6→1/5→…)을 화면에 표시. 서스펜스는 시도마다 증가.
- **악어 이빨 누르기** — 12개 중 함정 1개(판마다 재배치), 걸린 이빨은 빨간색으로 공개.
- **통아저씨** — 칼이 꽂힌 만큼 가려지는 마스크 연출, 트리거 구멍은 빨간색으로 공개.

### 🍟 이스터에그

- 감자튀김은 모든 시간대 룰렛에 항상 존재(노란 칸 전용). 당첨 시 🍟 파티클 폭발.
- **10회 보증(공개 규칙)**: 10번째 스핀은 감자튀김 확정 — 9번째부터 칸이 ✨ 반짝이며 예고.
- 숨은 치트: 감자튀김 라벨 더블탭.
