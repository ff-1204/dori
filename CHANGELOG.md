# Changelog

## v0.4.0 — 2026-07-18

### 🎯 랜덤 핀볼 대개편

- **매판 다른 랜덤 맵**: 핀 8행이 매번 새로 배치된다 — '🔀 맵 섞기' 버튼으로 핀 배치와 결과 칸 순서를 그 자리에서 다시 굴릴 수 있다(전부 보이는 상태에서 바뀜 — 정직).
- **칸 수 설정(4–10)**: 편집 패널의 ＋/−로 결과 칸을 조절 — 칸 수에 맞춰 공·핀 크기와 핀 밀도가 함께 조정된다.
- **조작 개선**: 공을 누른 채 아래로 끌어 화면 어디서 놓아도 낙하한다.
- **끼임 안전망**: 공이 구석에 멈추면 화면에 보이는 '틸트'로 빼내고, 그래도 안 되면 그 자리 기준으로 결과를 확정한다.

### 🔊 효과음 전면 배치

- 모든 게임에 하나의 루프로 배치: 출발 팝 → 빌드업 틱(룰렛 칸 통과·사다리 모서리·뽑기 상자 덜컹·핀볼 핀 히트) → 결과에 맞는 마무리 → 세이프는 안도 팝.
- **꽝 전용 하강음** 신설 — 꽝에도 승리 팡파르가 울리던 어색함을 제거.
- 소리 크기를 통일해 빵/쾅만 튀지 않게 조정. 효과음은 허브의 🔊 토글로 켠다(기본 꺼짐).

### 🎡 메뉴 룰렛

- **원하는 메뉴를 더블탭**하면 그 메뉴로 확정 스핀(숨은 기능 — 감자튀김 전용에서 전체 메뉴로 확장).
- 긴 메뉴 이름은 칸에 맞게 글자가 자동으로 작아진다.

### 🎭 연출 다듬기

- 통아저씨: 발사되면 눈이 X로 바뀐다(기절 연출).
- 악어 이빨: 물 때 이빨이 입 안에 자연스럽게 가려지고, 잠시 물었다가 입을 다시 열며 빨간 함정 이빨을 공개한다.
- 단청: 결과가 다시 묻기 전까지 유지되고, 붉·청 색패 표현으로 정돈.
- 허브 범주 라벨의 시각 위계 정리(작게·흐리게 — 게임 버튼이 주인공).

### 🛡 안정성

- 버튼 연타·동시 탭으로 화면이 겹치거나 멈추던 문제 수정(씬 전환 가드).
- 하단 툴바형 브라우저(네이버 웨일 등)에서 하단 바가 잘리던 문제 수정(동적 뷰포트 높이).
- 핀볼 공이 구석에서 무한히 튕기거나 게임이 멈추던 문제 수정.

### 📄 기타

- 사이트에 AI 학습 거부 신호(noai) 추가.

## v0.3.0 — 2026-07-16

### 🎮 신규 게임: 로또 번호 추첨

- 1–45에서 6개, **최대 5줄**(A–E) — 줄 수 조절 가능. 편향 없는 Fisher-Yates 셔플.
- **추첨은 매 정각 한 번** — 뽑은 번호는 다음 정각까지 잠금·기록되어 유지된다.
- **복사·공유 버튼**: 번호를 클립보드로, 또는 공유 시트로 바로 전달.
- 공 색은 공식 로또 색 구간(노랑·파랑·빨강·회색·초록)을 팔레트로 매핑.
- **"모든 조합의 확률은 같아요"를 상시 고지** — 확률 향상·당첨 보장류 표현 없음, 재추첨 시각은 정적 표기(카운트다운 없음).

### 🧭 허브 개편

- 게임 목록을 **2열 그리드**로 재배치 — 9종 라인업 수용, 이후 확장 대응.

### 📄 문서

- 개발 워크플로·확립 규칙을 [docs/workflow.md](docs/workflow.md)로 기록(파이프라인·실수→규칙).

## v0.2.0 — 2026-07-16

### 🎮 신규 게임: 단청(丹靑)

- 양자택일 오라클 — 질문을 입력하면 **붉(丹)** 또는 **청(靑)**으로 답한다.
- **결정적 해시**(FNV-1a + KST 4시간창): 서버 없이 어떤 기기·브라우저에서 물어도 **같은 질문 = 같은 답**, 답은 4시간(KST 정각 경계)에 1번만 바뀜 — 다시 물어도 번복되지 않음.
- 입력은 클라이언트에서만 처리(전송 없음). 카운트다운 없이 규칙만 차분히 안내.

### 📲 바로가기 (PWA)

- 허브 하단 **📲 바로가기** 버튼: 지원 브라우저(Chrome·Edge·안드로이드)는 즉시 설치 프롬프트, iOS는 방법 안내 모달.
- 매니페스트·서비스워커·자체 제작 앱 아이콘(룰렛 심볼) 구성 — 홈 화면/바탕화면에서 앱처럼 실행.
- 하단 바 재배치: 효과음 토글(좌) · 크레딧(중앙) · 바로가기(우).

### 🛡️ 법적·상업 보강

- **사용 제한 고지**: 오락 목적 전용 — 도박·사행·비윤리적 사용 금지를 LICENSE(조항)·README·사이트에 명시.
- 법적 검토 반영: 게임 명칭 상표 확인·게임물 등급분류를 수익화 전 체크리스트에 추가, MIT 고지 보존 방식 명문화, 개인정보 현황(수집 0) 기록.

### ✨ 다듬기

- 공유 이미지·검색 문구를 8종 라인업으로 갱신, 릴리스 태깅 규칙 문서화.

### 🎮 게임 7종

- **메뉴 룰렛** — 시간대(아침 05–11 / 점심 11–17 / 저녁 17–21 / 야식 21–05)에 맞는 트렌드 기반 메뉴 세트. 메뉴 추가·삭제(시간대별 독립 저장)와 기본값 복원, 숫자 룰렛 모드(1–N). 결과-우선 회전으로 포인터 칸과 결과가 항상 일치.
- **사다리타기** — 참가자 2–6명·결과 편집, 용도 프리셋(커피/벌칙/청소/당첨). '시작' 시 다리가 전부 공개되고 참가자 색으로 경로가 그려짐.
- **랜덤 핀볼** — 낙하 지점을 직접 고르고(주도성) 실제 물리로 낙하. 핀 히트 카운터, 막판 슬로모+줌, 착지 파티클. 칸 라벨 편집(기본 1·2·3등+꽝).
- **뽑기 상자** — 비복원 추첨. 뽑힌 항목은 흐려져 남은 풀이 항상 보임. 항목 2–12개 편집.
- **러시안 룰렛** — 6약실, 시도마다 실제 확률(1/6→1/5→…)을 화면에 표시. 서스펜스는 시도마다 증가.
- **악어 이빨 누르기** — 12개 중 함정 1개(판마다 재배치), 걸린 이빨은 빨간색으로 공개.
- **통아저씨** — 칼이 꽂힌 만큼 가려지는 마스크 연출, 트리거 구멍은 빨간색으로 공개.
