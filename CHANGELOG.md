# Changelog

## v0.7.0 — 2026-07-18

### 🌐 공식 도메인 dori.io.kr

- 사이트가 **https://dori.io.kr** 로 이사했다 — 기존 github.io 주소는 새 주소로 자동 연결되므로 저장해 둔 링크도 그대로 열린다.
- QR·주소 복사·조 편성 공유 링크·검색 미리보기 이미지가 모두 새 주소 기준으로 나간다.
- http로 들어와도 https로 안전하게 연결되고, 도메인 소유 검증까지 마쳐 탈취 위험을 막았다.

### 📄 소개·안내 페이지

- **소개**(about) — dori가 무엇인지, '화면에 보이는 그대로 뽑는 정직한 랜덤' 원칙, 운영자·문의처.
- **게임 안내**(guide) — 게임 10종의 방법·규칙·이럴 때 쓰세요를 한 페이지에.
- **개인정보처리방침**(privacy) — 수집하는 개인정보 없음, 기기 안에만 저장되는 항목 전체 목록과 삭제 방법, 광고 도입 시 바뀌는 점 예고(영문 요약 포함).
- 허브 하단에서 세 페이지로 바로 이동할 수 있다.

### 🎴 단청

- 모바일에서 질문을 입력한 뒤 화면을 탭하거나 '묻기'를 누르면 **자판이 내려간다** — 답 공개 연출이 가려지지 않는다.
- 자판의 '이동' 키로 바로 물을 수 있다.

### 🛡 상업 준비

- 광고(Google) 게재를 위한 심사 준비를 마쳤다 — **아직 광고는 없으며**, 도입 시 처리방침 갱신과 함께 알린다.
- 유럽 방문자용 동의 메시지(거부 버튼을 첫 화면에 노출하는 3버튼 구성)를 준비했다.

## v0.6.0 — 2026-07-18

### 🎮 신규 게임: 조 배정 (결정 돕기 10번째 게임)

- **번호 또는 이름으로 N조 분할**: 인원(2–60)·조 수(2–6)를 정하면 섞어서 공정하게 나눈다 — 안 나눠떨어지면 앞 조부터 1명 더(공개 규칙).
- **칩 클러스터 연출**: 번호·이름이 칩으로 하나씩 조 패널에 통통 모여든다(도착 순서 그대로 — 정직). 인원이 많으면 칩이 자동으로 작아진다.
- **명단 관리(버퍼)**: 이름을 등록하고 사람마다 탭으로 **참여 → 👑 조장 → 쉬기**를 순환 — 쉬는 사람은 명단에 남긴 채 이번 배정에서만 빠진다.
- **조장 지정**: 지정한 조장은 셔플 후 조에 1명씩 우선 배치(조장이 조 수보다 적어도 되고, 어느 조가 받을지도 랜덤). 지정이 없으면 '조장 뽑기' 토글로 첫 도착이 조장.
- **명단 복사·붙여넣기**: 조장/참여/쉼 제목 아래 이름을 나열하는 순수 텍스트로 내보내고, 같은 형식을 붙여넣어 통째로 교체(빈 명단이면 채워 쓸 양식이 복사됨).
- **조 편성 복사**: 결과를 조별 텍스트(조장 👑 표시)로 클립보드에 복사.
- **지난 편성 기억**: 마지막 결과가 기기에 저장되어 다시 들어와도 그대로 보인다(서버 전송 없음).

### 🎡 메뉴 룰렛

- 🍟 **감자튀김은 '나온 건 제외'에서도 영원하다** — 당첨돼도 제외되지 않고 항상 후보로 남는다(삭제 불가와 같은 규칙).
- 결과 후에도 버튼 라벨을 '돌리기'로 고정.

## v0.5.0 — 2026-07-18

### 🎡 메뉴 룰렛: 비복원 모드

- **'나온 건 제외' 토글**: 켜면 나온 메뉴 칸이 흐려지며 다음 스핀에서 빠진다(남은 풀이 항상 보이는 정직한 방식). **'↺ 제외 취소'**로 한 번에 복원.
- 제외 목록과 토글은 **기기 안에만 저장**되어 다시 들어와도 이어진다(시간대별 독립, 서버 전송 없음).
- **'⟲ 전체 초기화'**: 아침·점심·저녁·야식 4개 시간대 메뉴를 한 번에 기본값으로 — 실수 방지를 위해 두 번 눌러 확정.

### 🧭 이동이 자연스러워짐

- **뒤로가기 통합**: 게임 화면에서 모바일 백 제스처·브라우저 뒤로가기를 누르면 사이트를 나가는 대신 **허브로 돌아온다**(허브에서는 평소처럼 나감). 인게임 ⬅ 버튼과 브라우저 스택이 항상 일치.
- **해시 딥링크**: `#roulette`처럼 주소로 특정 게임에 바로 진입 — 게임 중 새로고침해도 그 게임 유지.
- 공유 버튼은 **주소 복사 전용**으로 단순화(라벨도 '주소 복사'로 정직하게).

### 💠 화면 마감

- **허브 개편**: 시간대별 인사말 **50종**(아침·점심·오후·저녁·밤 × 10) — 매시간 다른 문구로 회전하고 같은 시간엔 모두 같은 문구. 타이틀·간격 정돈, 인사말과 버튼의 은은한 등장 연출.
- **전 화면 공통 마감**: 씬 전환 페이드, 버튼 상단 광택(입체감), 하단 비네트(깊이), 모달·편집 패널 등장 페이드 통일, 토스트를 알약형 카드로.
- 뒤로가기 버튼을 ⬅ 아이콘만으로 축약(터치 영역은 그대로 크게), 넓은 화면의 레터박스에 비네트 배경.

### 🎴 단청

- **마지막 질문 기억**: 다시 들어오면 입력창에 지난 질문이 미리 채워진다(기기 안에만 저장, 전송 없음).

### 📄 소개·공유

- 검색·공유 문구를 친근하게 다듬고("결정이 필요한 순간, 미니게임이 대신 골라드려요"), 공유 미리보기 이미지를 새로 제작.

## v0.4.0 — 2026-07-18

### 🎯 랜덤 핀볼 대개편

- **매판 다른 랜덤 맵**: 핀 8행이 매번 새로 배치된다 — '🔀 맵 섞기' 버튼으로 핀 배치와 결과 칸 순서를 그 자리에서 다시 굴릴 수 있다(전부 보이는 상태에서 바뀜 — 정직).
- **칸 수 설정(4–10)**: 편집 패널의 ＋/−로 결과 칸을 조절 — 칸 수에 맞춰 공·핀 크기와 핀 밀도가 함께 조정된다.
- **조작 개선**: 공을 누른 채 아래로 끌어 화면 어디서 놓아도 낙하한다.
- **끼임 안전망**: 공이 구석에 멈추면 화면에 보이는 '틸트'로 빼내고, 그래도 안 되면 그 자리 기준으로 결과를 확정한다.

### 🔊 효과음 전면 배치

- 모든 게임에 하나의 루프로 배치: 출발 팝 → 빌드업 틱(룰렛 칸 통과·사다리 모서리·뽑기 상자 덜컹·핀볼 핀 히트) → 결과에 맞는 마무리 → 세이프는 안도 팝.
- **꽝 전용 하강음** 신설 — 꽝에도 승리 팡파르가 울리던 어색함을 제거.
- 소리 크기를 통일해 빵/쾅만 튀지 않게 조정. 효과음은 허브의 🔊 토글로 켠다(기본 꺼짐).

### 🎡 메뉴 룰렛

- **원하는 메뉴를 더블탭**하면 그 메뉴로 확정 스핀(숨은 기능 — 감자튀김 전용에서 전체 메뉴로 확장).
- 긴 메뉴 이름은 칸에 맞게 글자가 자동으로 작아진다.

### 🎭 연출 다듬기

- 통아저씨: 발사되면 눈이 X로 바뀐다(기절 연출).
- 악어 이빨: 물 때 이빨이 입 안에 자연스럽게 가려지고, 잠시 물었다가 입을 다시 열며 빨간 함정 이빨을 공개한다.
- 단청: 결과가 다시 묻기 전까지 유지되고, 붉·청 색패 표현으로 정돈.
- 허브 범주 라벨의 시각 위계 정리(작게·흐리게 — 게임 버튼이 주인공).

### 🛡 안정성

- 버튼 연타·동시 탭으로 화면이 겹치거나 멈추던 문제 수정(씬 전환 가드).
- 하단 툴바형 브라우저(네이버 웨일 등)에서 하단 바가 잘리던 문제 수정(동적 뷰포트 높이).
- 핀볼 공이 구석에서 무한히 튕기거나 게임이 멈추던 문제 수정.

### 📄 기타

- 사이트에 AI 학습 거부 신호(noai) 추가.

## v0.3.0 — 2026-07-16

### 🎮 신규 게임: 로또 번호 추첨

- 1–45에서 6개, **최대 5줄**(A–E) — 줄 수 조절 가능. 편향 없는 Fisher-Yates 셔플.
- **추첨은 매 정각 한 번** — 뽑은 번호는 다음 정각까지 잠금·기록되어 유지된다.
- **복사·공유 버튼**: 번호를 클립보드로, 또는 공유 시트로 바로 전달.
- 공 색은 공식 로또 색 구간(노랑·파랑·빨강·회색·초록)을 팔레트로 매핑.
- **"모든 조합의 확률은 같아요"를 상시 고지** — 확률 향상·당첨 보장류 표현 없음, 재추첨 시각은 정적 표기(카운트다운 없음).

### 🧭 허브 개편

- 게임 목록을 **2열 그리드**로 재배치 — 9종 라인업 수용, 이후 확장 대응.

### 📄 문서

- 개발 워크플로·확립 규칙을 [docs/workflow.md](docs/workflow.md)로 기록(파이프라인·실수→규칙).

## v0.2.0 — 2026-07-16

### 🎮 신규 게임: 단청(丹靑)

- 양자택일 오라클 — 질문을 입력하면 **붉(丹)** 또는 **청(靑)**으로 답한다.
- **결정적 해시**(FNV-1a + KST 4시간창): 서버 없이 어떤 기기·브라우저에서 물어도 **같은 질문 = 같은 답**, 답은 4시간(KST 정각 경계)에 1번만 바뀜 — 다시 물어도 번복되지 않음.
- 입력은 클라이언트에서만 처리(전송 없음). 카운트다운 없이 규칙만 차분히 안내.

### 📲 바로가기 (PWA)

- 허브 하단 **📲 바로가기** 버튼: 지원 브라우저(Chrome·Edge·안드로이드)는 즉시 설치 프롬프트, iOS는 방법 안내 모달.
- 매니페스트·서비스워커·자체 제작 앱 아이콘(룰렛 심볼) 구성 — 홈 화면/바탕화면에서 앱처럼 실행.
- 하단 바 재배치: 효과음 토글(좌) · 크레딧(중앙) · 바로가기(우).

### 🛡️ 법적·상업 보강

- **사용 제한 고지**: 오락 목적 전용 — 도박·사행·비윤리적 사용 금지를 LICENSE(조항)·README·사이트에 명시.
- 법적 검토 반영: 게임 명칭 상표 확인·게임물 등급분류를 수익화 전 체크리스트에 추가, MIT 고지 보존 방식 명문화, 개인정보 현황(수집 0) 기록.

### ✨ 다듬기

- 공유 이미지·검색 문구를 8종 라인업으로 갱신, 릴리스 태깅 규칙 문서화.

### 🎮 게임 7종

- **메뉴 룰렛** — 시간대(아침 05–11 / 점심 11–17 / 저녁 17–21 / 야식 21–05)에 맞는 트렌드 기반 메뉴 세트. 메뉴 추가·삭제(시간대별 독립 저장)와 기본값 복원, 숫자 룰렛 모드(1–N). 결과-우선 회전으로 포인터 칸과 결과가 항상 일치.
- **사다리타기** — 참가자 2–6명·결과 편집, 용도 프리셋(커피/벌칙/청소/당첨). '시작' 시 다리가 전부 공개되고 참가자 색으로 경로가 그려짐.
- **랜덤 핀볼** — 낙하 지점을 직접 고르고(주도성) 실제 물리로 낙하. 핀 히트 카운터, 막판 슬로모+줌, 착지 파티클. 칸 라벨 편집(기본 1·2·3등+꽝).
- **뽑기 상자** — 비복원 추첨. 뽑힌 항목은 흐려져 남은 풀이 항상 보임. 항목 2–12개 편집.
- **러시안 룰렛** — 6약실, 시도마다 실제 확률(1/6→1/5→…)을 화면에 표시. 서스펜스는 시도마다 증가.
- **악어 이빨 누르기** — 12개 중 함정 1개(판마다 재배치), 걸린 이빨은 빨간색으로 공개.
- **통아저씨** — 칼이 꽂힌 만큼 가려지는 마스크 연출, 트리거 구멍은 빨간색으로 공개.

### 🍟 이스터에그
